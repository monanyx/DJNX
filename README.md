# DJ Deck Simulator

The DJ Deck Simulator is a remarkable application that allows users to experience the thrill of DJing with two realistic turntables featuring vinyl records. This simulator offers an immersive experience with the following features:

- **Realistic Turntables**: Enjoy the authentic look and feel of traditional turntables with detailed graphics.
- **Pitch Control**: Adjust the pitch of your tracks seamlessly for smooth transitions and creative mixing.
- **Scratching**: Utilize various scratching techniques just like a professional DJ.
- **Audio Playback**: Play high-quality audio tracks with ease.
- **Realistic Physics Simulation**: The simulator incorporates advanced physics to replicate real-world DJing, including:
  - **Motor Torque**: Mimics the force required to start and stop the turntables.
  - **Platter Inertia**: Provides a realistic response that mirrors actual vinyl behavior while spinning.
  - **Strobe Effects**: Enjoy visual effects that enhance the DJing experience, simulating real-life performance environments.

This DJ Deck Simulator is designed for both novice and experienced DJs, making it a perfect tool for practicing and honing your skills.
